# AVISHKAR 2026 – Task 1
Black-Box Model Behavior Analysis

## Description
Students must analyze the behavior of a pretrained AI model using only input-output observations.

## Setup
```
pip install -r requirements.txt
```

## Run
```
task1_inference.ipynb
```

## Rules
- Do NOT retrain the model.
- Do NOT modify architecture.
- Internet allowed only for setup.
- AI tools are not allowed during assessment.
- Evaluation focuses on reasoning and explanation.

## Expected Analysis
- Prediction behavior
- Error pattern observation
- Confidence analysis
- Model strengths and weaknesses
